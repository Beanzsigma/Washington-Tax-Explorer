# Washington Tax Explorer

VIDEO DEMO IS INCLUDED IN REPOSITORY 
A Python based Washington State tax calculator that computes sales tax with customizable rates and clean output formatting.
The Washington Tax Explorer is a calculator that estimates different types of taxes through an interactive desktop interface. This app includes a sales tax calculator, a capital gains tax estimator with threshold-based logic, and a property tax tool that uses county based rates. Users can input values such as income, purchase amount, etc, and the program calculates the tax and total cost. The project is designed as a modular multi tool system, where each tax type has its own section, making it easy to expand to more tax models in the future. 
Small amounts of AI is used to debug/fix code.
FYI: The web playable version of this code is converted from python to HTML. 

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python dashboard.py
