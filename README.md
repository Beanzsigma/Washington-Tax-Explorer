# Washington Tax Explorer
A Python based Washington State tax calculator that computes sales tax with customizable rates and clean output formatting.
The Washington Tax Explorer is a calculator that estimates different types of taxes through an interactive desktop interface. This app includeds a sales tax calcualtor, a capital gains tax estimator with threshold-based logic, and a property tax tool that uses county based rates. Users can input values such as income, purchase amount, etc, and the program calculates the tax and total cost. The project is designed as a modular multi tool system, where each tax type has its own section, making it easy to expand to more tax models in the future.
